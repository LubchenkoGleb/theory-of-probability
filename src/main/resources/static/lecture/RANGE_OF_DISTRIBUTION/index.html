Математичне сподівння

Математичним сподіванням M X( ) дискретної випадкової величини X називається сума добутків всіх її можливих значень на ймовірності, з якими випадкова величина приймає ці значення:

$$\text{M}[X] = \sum_{x \in \mathcal{X}}xP(x)$$

Розгляньмо експеримент з підкиданням кісток.

Дисперсія

Дисперсією (розсіянням) D X( ) випадкової величини X називається математичне сподівання квадрата її відхилення:,

$$\text{D}(X) = \text{M}[(X - \text{M}[X])^2]$$

Намалюйте випадкові картки з колоди з десяти карт. Зауважте, що середнє значення квадратів відмінностей (у зеленому кольорі) починає нагадувати дисперсію (синю).
